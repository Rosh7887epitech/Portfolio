# SpaceDuckShoot - Jeu de tir spatial

## Description
Un jeu de tir spatial mettant en scène des canards, développé pour illustrer la programmation de jeux en C++.

## Technologies utilisées
- Langage : C++
- Bibliothèque : SFML

## Installation et exécution
```bash
cd SpaceDuckShoot
make
./spaceduck
```

## Fonctionnement
Le joueur contrôle un vaisseau qui doit abattre des vagues de canards envahisseurs.

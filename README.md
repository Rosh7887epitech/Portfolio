##### 🐚 42sh
**FR:** Shell UNIX complet avec support des pipes, redirections, et fonctionnalités avancées. Implémentation en C avec gestion des signaux et de l'historique des commandes.

**EN:** Complete UNIX shell with pipes, redirections, and advanced features support. C implementation with signal handling and command history management.

**Technologies:** C, UNIX System Calls
**Features:** Pipes, Redirections, Built-in commands, Signal handling

##### 🔍 BSQ (Biggest Square)
**FR:** Algorithme optimisé pour trouver le plus grand carré vide dans une grille contenant des obstacles. Utilise la programmation dynamique pour une complexité temporelle optimale.

**EN:** Optimized algorithm to find the biggest empty square in a grid containing obstacles. Uses dynamic programming for optimal time complexity.

**Technologies:** C, Dynamic Programming
**Features:** 
- 📊 Grid parsing and validation
- 🧮 Dynamic programming algorithm
- 🎯 Optimal square detection
- 📝 Result visualization

##### ⚙️ Assembly
**FR:** Collection de fonctions standard de la libc réimplémentées en langage assembleur x86-64. Optimisations pour les performances et respect des conventions d'appel.

**EN:** Collection of standard libc functions reimplemented in x86-64 assembly language. Performance optimizations and calling convention compliance.

**Technologies:** Assembly x86-64, NASM
**Functions Implemented:**
- 📏 `strlen` - String length calculation
- 📋 `strcpy`, `strncpy` - String copying
- 🔍 `strcmp`, `strncmp` - String comparison
- 🧠 `memset`, `memcpy`, `memmove` - Memory operations
- 🔎 `strchr`, `strrchr` - Character searching
